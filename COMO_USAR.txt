COMO USAR O ANT'S 🐜
════════════════════════════════════════

O Ant's é uma colônia de bots com "mente colmeia": eles trabalham
juntos, lembram do que aprenderam e até criam aplicativos para você.


🌐 PELO NAVEGADOR (jeito mais fácil)
────────────────────────────────────────
1. Acesse:  [URL_DO_DEPLOY]
   (ou http://localhost:8765 se estiver rodando no seu computador)
2. Comece a conversar com a colônia — não precisa instalar nada.
   Dica: a página /acesso.html é uma entrada simples com um botão.


📱 NO CELULAR (como app)
────────────────────────────────────────
1. Abra o link no Chrome (Android) ou Safari (iPhone).
2. Toque em "Instalar" ou "Adicionar à tela de início".
3. O Ant's aparece como app na tela inicial!


💻 NO COMPUTADOR
────────────────────────────────────────
• Como site:  abra  [URL_DO_DEPLOY]
• Como app instalável:  Chrome → Menu → "Instalar Ant's"
• Rodando local:  bash scripts/start.sh  →  http://localhost:8765
• App nativo (offline):  bash scripts/build_app.sh


☁️ COLOCAR NA NUVEM (grátis, link 24/7)
────────────────────────────────────────
1 clique (recomendado): use o botão "Deploy" do Render/Railway no
README (secção "Deploy na nuvem").
Ou por CLI:  bash scripts/deploy_cloud.sh  (Railway, Render ou Fly.io).
Ao terminar, a plataforma te dá uma URL pública — cole em [URL_DO_DEPLOY].


❓ DÚVIDAS
────────────────────────────────────────
• Precisa de internet?     Sim, para acessar a nuvem.
• Meus dados estão seguros? Sim, processados no servidor que você controla.
• Posso usar offline?       Sim, instalando o app nativo (Tauri).
